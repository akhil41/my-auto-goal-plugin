---
name: my-auto-goal
description: Start an autonomous goal-achievement loop for research, code, app, content, or web-seo work
trigger: command
---

This command activates the my-auto-goal skill.

When `/my-auto-goal` is invoked, immediately use the Skill tool:

```
Skill({ skill: "my-auto-goal-plugin:my-auto-goal" })
```

The skill will guide goal clarification, get approval, and run the autonomous loop.
